# Policy Gradient Methods on CartPole-v1

This repository contains the implementation and comparative analysis of three fundamental policy-gradient algorithms: **REINFORCE**, **Actor-Critic (AC)**, and **Advantage Actor-Critic (A2C)**. The project evaluates these methods against a **Deep Q-Network (DQN)** baseline to investigate the practical implications of the bias-variance trade-off and training stability in Reinforcement Learning.
